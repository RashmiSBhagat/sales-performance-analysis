# Sales Performance Analysis Dashboard

## Objective
Analyze sales data to identify trends, patterns, and insights to support business decision-making.

## Tools Used
- Excel
- SQL
- Power BI

## Dataset
Sales data containing order details, region, product, sales, quantity, and profit.

## Key Analysis
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Regional and monthly sales trends
- Product-wise profitability

## Dashboard
The Power BI dashboard presents:
- Total sales KPIs
- Regional performance comparison
- Monthly sales trends
- Product profit analysis

## Key Insights
- North and East regions show higher revenue contribution
- Laptops generate the highest profit margin
- Sales show an upward trend over months

## Outcome
This project demonstrates end-to-end data analysis skills including data preparation, visualization, and insight communication.
